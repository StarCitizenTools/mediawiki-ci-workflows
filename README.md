# mediawiki-ci-workflows

Reusable GitHub Actions CI workflows for MediaWiki skins and extensions.

## Workflows

| Workflow | Purpose |
|----------|---------|
| `lint.yml` | PHP, JS, styles, i18n linting |
| `test-php.yml` | PHPUnit against MediaWiki matrix |
| `test-js.yml` | Vitest with coverage |
| `analyze-php.yml` | Phan static analysis |
| `sonarqube.yml` | SonarQube scan with coverage |

## Usage

See each workflow file for available inputs and examples.
